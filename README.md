#  Password Generator (Python)

A simple Python-based password generator that creates secure, random passwords using letters, numbers, and special characters.  
This is a beginner-friendly project and a great starting point for learning Python.

---

## Features

- Generates a random password
- Includes uppercase, lowercase, digits, and punctuation
- Customizable password length
- Easy to run in any environment

---

## Requirements

No external libraries are required.

Just make sure you have **Python 3.6+** installed.
